## 🚀 如何游玩

### 方法一：一键启动（推荐）

**双击 `启动游戏.bat`**，脚本会自动启动本地服务器并打开浏览器。

> 需要电脑安装 **Python 3** → [python.org](https://www.python.org/downloads/)
> 安装时勾选 "Add Python to PATH"

### 方法二：命令行启动

```bash
# 在游戏文件夹中打开命令行，输入：
python -m http.server 21777
# 浏览器打开 http://localhost:21777
```

### ❓ 不能直接双击 HTML

现代浏览器安全策略禁止从 `file://` 加载模块化脚本，必须通过本地服务器运行。

## 📖 游戏简介

你叫林墨，23岁，入职了一家叫"普罗米修斯"的生物科技公司。
从第一天起，诡异的事情接连发生——发酵罐异常、DNA库被入侵、同事离奇死亡。
更可怕的是——每次死亡后你都会回到入职第一天的早上。

你发现了一种叫 **QZ-酶** 的神秘蛋白质——它能在宿主濒死时触发"意识回溯"。
而你的DNA中恰好携带激活它的突变——你是"完美适配体"。

| 角色 | 身份 | 生化主题 |
|------|------|---------|
| 🔵 林雪 | 结构生物学家 | 蛋白质结构、晶体学 |
| 🟠 陈阳 | 有机化学家 | 糖代谢、有机合成 |
| 🔴 苏菲亚 | 分子生物学家 | 线粒体、氧化磷酸化 |
| 💗 小雅 | AI·生物信息学 | 生物信息学、RNA世界 |
| 🟢 王博士 | 酶学家 | 酶动力学、代谢调控 |
| 🟣 张总 | CEO·隐藏路线 | 端粒、衰老机制 |

**规模：** ~3500行剧本 · 64个知识问答 · 76个知识条目 · 12张CG · 16个结局
